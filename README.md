# My Pilot Plan

A static web dashboard for glider and powered flight preparation.

## How to publish to GitHub

1. Create a new repository on GitHub named `My_Pilot_Plan`.
2. Add it as a remote in this folder:

   ```powershell
   git remote add origin https://github.com/<your-username>/My_Pilot_Plan.git
   git branch -M main
   git push -u origin main
   ```

3. In GitHub repository settings, enable GitHub Pages from the `main` branch root.

Once published, your site will be available at:

```text
https://<your-username>.github.io/My_Pilot_Plan/
```
